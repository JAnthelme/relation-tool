Other operators

semiJoin :: Relvar -> Relvar -> Relvar Source

Similar to the join operator projected over the attributes of the first relation. Loosely speaking the results of semiJoin r1 r2 are the rows of r1 which have a counterpart in r2.

r1 `semiJoin` r2 == project (r1 `join` r2) (labels r1)

semiMinus :: Relvar -> Relvar -> Relvar Source

Similar to the minus operator projected over the attributes of the first relation. Loosely speaking the results of semiMinus r1 r2 are the rows of r1 which do not have a counterpart in r2.

r1 `semiMinus` r2 == project (r1 `minus` r2) (labels r1)
r1 `semiMinus` r2 == r1 `minus` (r1 `semiJoin` r2)

summarize :: Relvar -> Relvar -> [OpElem] -> [String] -> [String] -> Relvar Source

summarize r per ops inlbls outlbls computes ops over inlbls columns into outlbls column.

group :: Relvar -> [String] -> String -> Relvar Source

group r grplbls reslbl returns a relation with all of r's attributes, excluding grplbls columns, plus reslbl column. The elements in reslbl column are of type Relvar and the result of grouping of grplbls columns over the non-grplbls of r.

degree (group r grplbls reslbl) == degree r - (length grplbls) + 1
